# JSON Schemas

Each pass emits a distinct JSON shape.

## Structure

```json
{
  "chapters": [],
  "parts": [],
  "scene_breaks": [],
  "status": ""
}
```

## Dialogue

```json
{
  "pov": "",
  "dialogue": false,
  "internal_thought": false,
  "explicitly_attributed_speakers": [],
  "unattributed_dialogue_present": false
}
```

## Entities

```json
{
  "characters": [],
  "places": [],
  "objects": [],
  "identity_notes": []
}
```

## Dossiers

```json
{
  "character_dossiers": []
}
```
